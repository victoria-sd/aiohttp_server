# **<span style="font-size:24px;">Реализация сервера для чата и новостей в реальном времени</span>**

Этот проект представляет собой сервер на базе Python и aiohttp, который обеспечивает работу веб-сокетов для обмена сообщениями и новостями в реальном времени. Клиенты могут подключаться через веб-браузер и получать обновления мгновенно, а также отправлять новости, которые будут рассылаются всем подключенным клиентам.

---

## **Основные возможности**
- Поддержка множественных веб-сокет клиентов
- Обмен сообщениями и новостями в реальном времени
- Возможность отправки новостей через POST-запрос
- Автоматическая проверка соединения с клиентами с помощью PING/PONG

---

## **Структура проекта**
- `server.py` — основной серверный скрипт
- `websocket.html` — HTML-страница для клиента
- `static/` — папка для статических файлов (CSS, JS)

---

## **Как запустить**

1. Установите необходимые зависимости:

```bash
pip install aiohttp
```

2. Запустите сервер:

```bash
python server.py
```

3. Откройте браузер и перейдите по адресу:

```plaintext
http://localhost:8080
```

---

## **Как отправить новость**

Для добавления новости в реальном времени необходимо выполнить POST-запрос. Например, через PowerShell:

```powershell
Invoke-RestMethod -Uri http://localhost:8080/news -Method POST -Headers @{"Content-Type"="application/json"} -Body '{"news": "New news!"}'
```

Это отправит новость "New news!" всем подключенным клиентам.
